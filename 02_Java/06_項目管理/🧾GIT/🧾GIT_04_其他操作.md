# GIT_04_其他操作
## 🧾垃圾清理
清理不必要的檔案並優化本地儲存庫：`git gc`
`git gc --force`，即使可能有另一个git gc实例在此存储库上运行，也强制运行。
删除在本地有但在遠程库中已經不存在的分支：`git prune `

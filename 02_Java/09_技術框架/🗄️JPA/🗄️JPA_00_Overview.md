# JPA
- JPA(Java Persistence API)：用於標準化ORM(Object Relational Mapping)框架的規范，使應用程序以統一的方式訪問持久層
	![[JPA_00_Overview_01_原理.png]]
- 包含技術
	- ORM映射數據
	- JPA API執行CRUD操作
	- JPQL查詢語言
